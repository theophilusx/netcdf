Read a value from an array map returned by a call to read-slice. The a argument is an array map returned from a call to read-slice. The function is defined to handle multiple index arguments, depending on the rank of the array. The function supports arrays with a rank from 0 to 7 dimensions. The dimension indices are specified as integers.
